Equity Tracker

Database Locked

No portfolio data can be loaded until the database is unlocked.

Recovery checklist

  1. Unlock with POST /admin/unlock if you are running in manual unlock mode.
  2. For auto-open mode, verify EQUITY_DB_PATH and EQUITY_DB_PASSWORD, then restart the server.
  3. Retry this page after unlock succeeds.

If unlock still fails, check the latest server logs for database open errors and path/password mismatch details.
